gitplug
=======
git plugin for ranger file manager.

#### Working command list:
* ```:git init```
* ```:git status```
* ```:git clone <url>```
* ```:git add <file>```
* ```:git rm <file>```
* ```:git restore <file>``` 
* ```:git remote add/rm <name> <url>```
* ```:git push```
* ```:git commit <text>```
